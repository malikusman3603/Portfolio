# Project Summary
Muhammad Usman Portfolio
A modern, responsive single-page website that showcases Muhammad Usman’s background, skills, education, work experience, and projects. It also provides an easy way for recruiters, collaborators, and academic contacts to learn more and get in touch.

Font Family: Poppins, system-ui, sans-serif

Body: 1rem (16px)
Button/Text Input: 1rem (16px)

# Features
Responsive navigation bar (anchors to sections)
Hero section with animated accent, CTA buttons, and profile image
About Me section with bio, additional CTA button
Education & Experience (“My Journey”) cards, clean split for clarity
Skills display (coding & professional) with progress bars for proficiency
Contact section with web form (Full Name, Email, Mobile, Subject, Message)
Sticky footer, copyright
Consistent dark theme and color accenting throughout
Accessibility (alt tags, contrast, keyboard navigation)
Fast loading, mobile-ready
